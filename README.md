# DesafioDio-IA_Que_Orienta

# 🚀 IA Que Orienta

Projeto desenvolvido como parte de um desafio da DIO, com foco na criação de uma Inteligência Artificial capaz de orientar usuários por meio de respostas inteligentes e contextualizadas.

---

## 📌 Sobre o Projeto

O **IA Que Orienta** é uma aplicação que utiliza conceitos de **IA generativa e engenharia de prompts** para fornecer sugestões, direcionamentos e respostas úteis ao usuário.

A proposta é ir além de respostas simples, entregando uma experiência onde a IA atua como um **guia**, ajudando na tomada de decisões e aprendizado.

---

## 🎯 Objetivo

- Aplicar conceitos de Inteligência Artificial  
- Trabalhar com engenharia de prompts  
- Criar um assistente orientador  
- Desenvolver um projeto prático para portfólio  

---

## 🧠 Funcionalidades

- 💬 Interpretação de perguntas do usuário  
- 🧭 Respostas com foco em orientação (não só informação)  
- ⚡ Geração dinâmica de conteúdo  
- 🎯 Sugestões baseadas no contexto fornecido  

---

## 🛠️ Tecnologias Utilizadas

- Python  
- APIs de IA (OpenAI, Gemini ou similares)  
- Engenharia de Prompt  
- Git e GitHub
